Detailed Description

Extra data attached to each symbol, stored in each node.

Stores all the extra data attached to symbols in a node. The class is used by the evaluation_function class. It also provides the sink functions determining when to ignore a node due to user-defined criteria, i.e. low frquency counts. Important: Each evaluation_data type has to be registered by calling REGISTER_DEF_DATATYPE(<unique name>).
